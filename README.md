# SmartDiet — Inteligência Nutricional

Documentação oficial para orientar o desenvolvimento do SmartDiet no VS Code com OpenAI Codex.

## Como usar no VS Code
1. Copie esta pasta para dentro do repositório do SmartDiet.
2. Abra o projeto no VS Code.
3. Abra o Codex.
4. Peça: `Leia AGENTS.md e toda a pasta docs antes de implementar qualquer coisa.`
5. Depois solicite uma tarefa específica, por exemplo: `Implemente o módulo de pacientes seguindo a documentação.`

## Ordem recomendada de implementação
1. Estrutura do monorepo.
2. Backend FastAPI mínimo.
3. Frontend Next.js com layout base.
4. Banco PostgreSQL com migrações iniciais.
5. Dashboard.
6. Pacientes.
7. Alimentos.
8. Receitas.
9. Anamnese e avaliações.
10. Plano alimentar e diário.

## Estado comercial
O SmartDiet será gratuito durante a fase Beta, mas deve ser desenvolvido com qualidade comercial desde o início.
