# Design System — SmartDiet

## Identidade
- Nome: SmartDiet
- Slogan: Inteligência Nutricional
- Logo: letra S integrada a uma folha.
- Estilo: SaaS premium, natural, limpo, profissional.

## Paleta
| Token | Hex | Uso |
|---|---|---|
| forest | #2F6F4F | Cor principal |
| sage | #A8D5BA | Apoio suave |
| terracotta | #C86A3A | Destaque, alertas leves, CTAs secundários |
| petrol | #2E5D66 | Informação e gráficos |
| background | #F8F8F4 | Fundo principal |
| graphite | #374151 | Texto principal |
| white | #FFFFFF | Superfícies |

## Tipografia
Preferência: Geist. Alternativa: Inter ou Manrope.

## Princípios visuais
- Muito espaço em branco.
- Bordas arredondadas suaves.
- Sombras discretas.
- Poucas cores por tela.
- Estados de hover leves.
- Sem aparência de sistema hospitalar antigo.

## Componentes principais
- AppShell
- Sidebar
- Header
- SmartCard
- MetricCard
- SmartSearch
- SmartTable
- PatientCard
- FoodCard
- RecipeCard
- MealCard
- NutritionBadge
- MacroChart
- Timeline
- Modal
- Drawer
- Toast

## Layout
- Desktop first com responsividade.
- Sidebar fixa à esquerda.
- Conteúdo em cards.
- Dashboard com cards e listas úteis, não excesso de gráficos.

## Plano alimentar
Deve ter no mínimo 6 refeições:
- Café da manhã.
- Lanche da manhã.
- Almoço.
- Lanche da tarde.
- Jantar.
- Ceia.

Cada refeição deve exibir:
- Horário.
- Alimentos.
- Quantidades.
- Calorias.
- Proteínas.
- Carboidratos.
- Gorduras.
- Fibras.
- Sódio quando relevante.
- Substituições.
- Observações clínicas.
