# Especificação — Pacientes

## Objetivo
Centralizar informações pessoais, clínicas e nutricionais do paciente.

## Dados principais
- Nome completo.
- Data de nascimento.
- Sexo/gênero.
- Telefone.
- Email.
- Observações.
- Status: ativo, inativo, acompanhamento pausado.

## Tela do paciente
Deve exibir abas:
- Resumo.
- Anamnese.
- Avaliações.
- Bioimpedância.
- Plano alimentar.
- Diário alimentar.
- Receitas indicadas.
- Relatórios.

## Resumo
- Último peso.
- IMC.
- Objetivo.
- Última consulta.
- Próxima ação.
- Alertas.
