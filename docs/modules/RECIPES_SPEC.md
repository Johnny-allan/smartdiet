# Especificação — Receitas

## Objetivo
Permitir criação de receitas profissionais com cálculo nutricional automático.

## Campos
- Título.
- Descrição.
- Ingredientes.
- Modo de preparo.
- Tempo de preparo.
- Tempo de cocção.
- Rendimento.
- Porções.
- Peso cru.
- Peso pronto.
- Perda por cocção.
- Imagem.
- Etiquetas.
- Observações profissionais.

## Cálculo automático
A receita deve calcular nutrientes a partir dos ingredientes e suas quantidades em gramas.

## Nutrientes principais
- Kcal.
- Proteínas.
- Carboidratos.
- Gorduras.
- Fibras.
- Sódio.

## Experiência visual
Receitas devem parecer cartões premium, com foto opcional, macros e etiquetas.
