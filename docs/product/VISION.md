# Visão do Produto — SmartDiet

## Nome
SmartDiet

## Slogan
Inteligência Nutricional

## Posicionamento
Plataforma web profissional para nutricionistas, clínicas e estudantes de nutrição, criada para simplificar nutrição clínica, prescrição dietética, acompanhamento de pacientes, avaliação física e gestão de alimentos/receitas.

## Fase atual
Versão 1.0 Beta gratuita.

## Fora do escopo inicial
- Aplicativo mobile.
- Pagamentos.
- Assinaturas.
- Marketplace.
- Financeiro completo.

Esses módulos devem ser previstos arquiteturalmente, mas não implementados agora.

## Objetivo principal da versão Beta
Permitir que nutricionistas testem fluxos essenciais:
- Dashboard.
- Pacientes.
- Anamnese.
- Avaliação física.
- Bioimpedância.
- Banco de alimentos.
- Banco de receitas.
- Plano alimentar.
- Lista de substituições.
- Diário alimentar.
- Relatórios básicos.

## Diferenciais esperados
- Interface premium, limpa e rápida.
- Busca inteligente de alimentos.
- Cálculo nutricional automático.
- Organização clínica completa.
- Preparado para bases públicas de alimentos.
- Arquitetura API-first.
- Pronto para crescer sem reescrita.
