# SmartDiet Frontend

Frontend Next.js da plataforma SmartDiet - Inteligencia Nutricional.

## Requisitos

- Node.js 22+

## Desenvolvimento local

```bash
npm install
npm run dev
```

URL local: `http://localhost:3000`

## Tokens visuais

As cores oficiais ficam em `tailwind.config.ts` e devem ser usadas via classes Tailwind:

- `forest`
- `sage`
- `terracotta`
- `petrol`
- `background`
- `graphite`
- `surface`

Componentes finais nao devem usar hex direto.
